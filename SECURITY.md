# Security Policy

## Reporting a vulnerability
- Please do **not** open a public issue for suspected vulnerabilities.
- Email `security@conduit.dev` with details, steps to reproduce, and any proof of concept. If email is unavailable, mark your GitHub issue as **SECURITY** and request it be made private.
- You will receive an acknowledgment within 3 business days.

## Supported versions
- Current release: 1.0.0
- Older versions: not supported.

## Coordination
- We will confirm, triage, and provide guidance on fixes or mitigations.
- Please allow reasonable time for a fix before public disclosure.
